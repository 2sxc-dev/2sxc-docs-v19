JSON Assets V1

JSON Data can include Assets. This is currently only used for export/import of Views

Assets

The Assets is optional on the Entity node. It looks like this:

  • Assets array, optional
    • asset object 1
      • Storage: "app"
      • Name: "some file name"
      • Folder: "DefaultTemplates" (location in the storage)
      • File: "string containing the file"
    • asset object 2
      • ...

Note that in a view, typically it will export both the template file and the icon used.
