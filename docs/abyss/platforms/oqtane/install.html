Upgrade Existing Installations

When upgrading an existing Oqtane/2sxc you should follow these steps:

  1. make sure that a compatible 2sxc exists
  2. If it is compatible to both old and new (eg v16.09) then install 2sxc first
  3. else upgrade Oqtane first - possibly 2sxc then stops working
  4. then upgrade 2sxc
