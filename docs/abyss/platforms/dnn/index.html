Dnn Platform

Dnn is the most popular open-source CMS platform on .net. It's been around since ca. 2001.

2sxc has been running on Dnn since it's inception in 2012.

Upgrade Path

We are very focused on making upgrades easy and pain-free. Here's what you need to know:

  1. You can upgrade 2sxc by installing the latest version on a system which already has 2sxc installed.
  2. If you have a version before 8.12, you must first upgrade to 8.12.
  3. In general when updating from v9 to v14 there are no expected problems.
  4. Before going to v14 you must be running DNN 9.6.1 or newer.
  5. In very rare cases you may be affected by breaking changes

Features currently not implemented

None, all 2sxc features are available in DNN.

History

  • 2sxc v1 was for Dnn 4 in 2012
  • 2sxc v7 - v13 all run on Dnn 7.4.2 up until the latest Dnn 9.x
  • 2sxc v14 requires Dnn 9.6.1 or newer
