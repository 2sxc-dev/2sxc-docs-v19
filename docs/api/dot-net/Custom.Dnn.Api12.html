Base Class for Api Controllers for Dnn ‚ò¢Ô∏è only.

This is the Dnn specific base class. It's the very same as Custom.Hybrid.Api12 except for the additional Dnn Property. If you inherit from this class, you'll automatically have all the properties from DynamicCode.

Because of this, the documentation here only shows Dnn - for all other properties go to Custom.Hybrid.Api12

Tip

If you don't need the Dnn object, we strongly suggest to move to the Hybrid implementation.

The latest Hybrid base classes also have many new features and are easier to use. See Custom.Hybrid for more information.
