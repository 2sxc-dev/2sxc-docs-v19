Interface SxcGlobal

This is the root global window.$2sxc function / object. It is both a function window.$2sxc(...) and object window.$2sxc.insights...

If the page feature 2sxc.JsCms is enabled, the window.$2sxc will also be a SxcGlobalWithCms

Main Function $2sxc(...)

This is a object/function hybrid. So it is usually used like this:

var sxc = window.$2sxc(moduleId);

The signature /parameters is the same as the $2sxc.get(...) function.

The resulting object is a Sxc
