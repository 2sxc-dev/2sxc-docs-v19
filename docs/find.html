Internal Notes / Information

Goal is to redirect to a documentation XREF or Namespace. If we ever make bigger changes which change important XREFs, we would add manual redirect XREFs to the /xrefmap.yml.

  • Shortlink: https://go.2sxc.org/find
  • Url parameter: xref
  • Example: https://go.2sxc.org/find?xref=ToSic.Eav.Data
